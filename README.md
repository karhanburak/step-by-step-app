# Step by Step - AI Görev Asistanı

Bu proje React Native ve Expo kullanılarak oluşturulmuştur.

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Uygulamayı başlatın:
```bash
npx expo start
```

## Özellikler

- **AI Destekli Görev Oluşturma**: Yapmak istediğiniz işi yazın, AI adımları oluştursun.
- **Adım Takibi**: Adımları işaretleyin, ilerlemenizi görün.
- **Not Ekleme**: Her adıma özel notlar ekleyin.
- **AI Yardım**: Takıldığınız yerde AI'dan yardım isteyin.
- **Geçmiş**: Tamamladığınız görevleri saklayın.

## API Anahtarı

`src/services/geminiService.js` dosyasında `API_KEY` değişkenine kendi Google Gemini API anahtarınızı ekleyin. Varsayılan olarak mock veri ile çalışır.
